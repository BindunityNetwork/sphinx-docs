マップスクリプト
==============

マップスクリプトの読み込み
マップスクリプトに関しては `こちら <http://pvp-docs.minecraft.jp/ja/latest/script/index.html>`_

Code <Single>
--------

.. code-block:: yaml

    script:
    - src: map.js

Code <Multiple> 
--------

.. warning:: 

    複数scriptの読み込みに対応しているかはわかりません。

.. code-block:: yaml

    script:
    - src: map.js
    - src: main.js
